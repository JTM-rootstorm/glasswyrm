# Milestone 14 compatibility scaffolding

This directory intentionally contains no promoted traces, images, timing
fixtures, or hardware-acceptance result. `validate_client_state.py` validates
only the deterministic state emitted by the repository-owned
`m14_vrr_client`; it explicitly does not turn client state into hardware proof.

Final fixtures may be generated only after the G2 integration freeze and the
reviewed hardware run.
